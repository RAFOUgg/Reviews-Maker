Raccourcis & Astuces

⌨️ Raccourcis clavier

  • Ctrl/⌘ + ↑ / ↓ : Changer de section
  • Ctrl/⌘ + Entrée : Générer la review
  • Tab / Shift + Tab : Navigation entre les champs
  • Échap : Fermer les modales

📝 Étapes pour compléter une review

  1. Choisir le type de produit (Hash, Fleur, Concentré)
  2. Remplir les sections affichées (les totaux se mettent à jour automatiquement)
  3. Générer la review pour prévisualiser et vérifier
  4. Exporter en PNG et/ou CSV (l'état « Brouillon » disparaît à l'export)

✨ Astuces

  • Les champs séquence permettent d'ajouter plusieurs étapes dans l'ordre (ex: Bubble Hash → Séparation électrostatique)
  • Ajoutez une photo pour un rendu optimal
  • La sauvegarde est automatique pendant la saisie
  • Vos brouillons sont accessibles depuis "Ma Bibliothèque"
